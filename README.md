# Фотохостинг gekkkko

Репозиторий для загрузки и работы сайта gekkkko.com, который стоит на сервере Render.com.

Ссылка на сайт: https://gekkkko.com/

# Функционал:

Расположение изображений в интернете.

Получение ссылки для открытия этих изображений.

Возможность загрузки множества изображений.

Возможность защиты изображений паролем.

Возможность накладывать число просмотров для одного изображния.

Редактирование размеров одного изображения.

Загрузка изображений в базу данных.
